# Project Plan

## Cel
Stworzenie aplikacji webowej do kontroli miesięcznych wydatków.

## Zakres MVP
1. Ustawienie miesięcznego budżetu.
2. Ustawienie limitów kategorii.
3. Dodawanie wydatków.
4. Edycja wydatków.
5. Usuwanie wydatków.
6. Historia wydatków.
7. Podsumowanie miesiąca.
8. Informacja o pozostałych środkach.
9. Ostrzeżenia o przekroczeniu limitów.
10. Podstawowe statystyki.

## Warstwy projektu
- baza danych PostgreSQL,
- backend REST API w FastAPI,
- frontend React + TypeScript.

## Priorytety
1. Database design
2. Backend logic
3. API
4. Frontend UI
5. Local setup
